# SitS-Analysis

This repository contains R scripts for analysing soil sensor data and field sampling data from the Signals in the Soils (SitS) project.

The tree R scripts beginning with "ColtPark" are associated with paper "Multiple targeted grassland restoration interventions enhance ecosystem service multifunctionality" (https://www.nature.com/articles/s41467-025-59157-8)

R scripts ** are associated with paper "A changepoint approach to automated estimation of soil moisture drydown parameters from time series data" (https://www.nature.com/articles/s41598-025-27067-w).
